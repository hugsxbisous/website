Embeddable Carousel Widget Generator

This page does two jobs. It previews the carousel widget and also helps someone generate the embed code for their own copy. They create a copy of the Google Sheet template, publish that sheet as CSV, paste the CSV URL here, and this page produces the iframe code they can place elsewhere.

Create your widget

The template sheet should have these exact headers in row 1: date, title, blurb, background image. The user still needs to use Google Sheets File → Share → Publish to web and choose CSV before the widget can load the data.

Click “Create widget code” to generate the iframe snippet.
The generated code uses query parameters, so one widget page can power many different widgets without duplicating the file.

How this flow works

Step 1. Click Open template sheet and make a Google Sheets copy.
Step 2. Fill in rows using date, title, blurb, and background image.
Step 3. Publish that sheet to the web as CSV.
Step 4. Paste the published CSV URL into this page and click Create widget code.
Step 5. Copy the generated iframe embed code into any site that supports embedded iframes.
